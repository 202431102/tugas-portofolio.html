# Portofolio Web Sederhana

## Deskripsi
Situs web portofolio pribadi yang dibuat sebagai tugas . Berisi halaman Home , Portfolio , dan Kontak . Situs responsif menggunakan Bootstrap 5.

## Fitur
- **Home**: Hero section dengan foto diri, deskripsi singkat, about me, dan skills dengan progress bars.
- **Portfolio**: Tabel responsif dengan 5 proyek dummy (nama, tahun, teknologi, status, link demo).
- **Kontak**: Form request project dengan input nama, email, select layanan, textarea deskripsi, radiobutton prioritas, checkbox syarat, dan submit button.
- Navigasi konsisten di semua halaman.
- Footer dengan link ke [ITPLN](https://itpln.ac.id) di setiap halaman.

## Teknologi yang Digunakan
- HTML5, CSS3, Bootstrap 5.
- Responsif untuk desktop dan mobile.
